# Electricity & Circuits Quiz

**An interactive 8th-grade science quiz covering resistors, Ohm's Law, circuits, electromagnets, and more.**

🔗 **[Play it live →](https://medusan31.github.io/Science-Study-Website)**

---

## What is this?

A fully browser-based quiz app built to help students study electricity and circuits at the 8th grade level. Every time you start a quiz, a random selection of questions is pulled from a bank of **106 questions** — so no two sessions are the same.

After finishing, you can enter **Review Mode** to go through every question and see not just the correct answer, but *why* each wrong answer is wrong — which is where the real learning happens.

---

## Features

- **106-question bank** — randomized every session so the quiz stays fresh
- **Adjustable length** — choose anywhere from 1 to 106 questions per session
- **Two question types:**
  - Multiple choice with 4 options
  - Calculation questions with a number input and unit dropdown (A / V / Ω / W)
- **Score screen** — animated score ring with correct/incorrect breakdown
- **Full review mode** — for every multiple-choice question, see:
  - Why the correct answer is right
  - Why someone might choose each wrong answer — and why that reasoning fails
- **Calculation review** — side-by-side your answer vs. the correct answer
- No account, no login, no install — runs entirely in the browser

## Topics Covered

| Category | Examples |
|---|---|
| Ohm's Law | V = IR, solving for V / I / R |
| Power formulas | P = IV, P = I²R, P = V²/R |
| Series circuits | Total resistance, current, voltage drops |
| Parallel circuits | Total resistance, voltage, current splits |
| Batteries | Series vs. parallel battery configurations |
| Electrical diagrams | Symbols for resistors, switches, bulbs, batteries |
| Conductors & insulators | Materials, free electrons, real-world examples |
| Electromagnets | Strength factors, on/off control, applications |
| Electromagnetic induction | Generators, Faraday's discovery |
| AC vs DC | Differences, sources, safety |
| Historical figures | Volta, Ohm, Ampère, Watt, Faraday, Edison, Tesla, Franklin, Kirchhoff, Ørsted |
| Safety & protection | Fuses, circuit breakers, short circuits, overloading |
